# pomodoro
time control app
**Pomodoro** is a cross-platform Pomodoro timer application built with Python.  
It features a GUI using Tkinter, a system tray icon with pystray, and hotkey support using keyboard.

---

# Features

- Adjustable work, short break, and long break durations
- Daily Pomodoro count tracking
- Dark mode support
- Runs in the system tray
- Hotkey to show the window (Ctrl+Alt+P)
